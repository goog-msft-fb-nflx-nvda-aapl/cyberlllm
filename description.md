# COURSE TERM PROJECT HANDOUT
## LLM Applications in Cybersecurity: Proof of Concept (PoC) Development

---

## 1. Project Overview & Learning Objectives

This term project offers an immersive opportunity to bridge the gap between Large Language Model (LLM) theories and practical implementations within the Cybersecurity domain. Teams are expected to build, evaluate, and document a robust Proof of Concept (PoC) application addressing a critical security challenge using modern LLM techniques.

By completing this project, you will demonstrate capabilities across two core learning objectives:

1. **Advanced Technical Application**: Successfully synthesize and deploy core techniques, model architectures, and prompt methodologies acquired throughout the course onto a meaningful, real-world scale security system.

2. **Independent Ideation & Research**: Formulate, justify, and design a novel application scope by comprehensively consulting the foundational concepts and architectures highlighted in the Week 12 lecture slides.

---

## 2. Core Scope & Acceptable Domains

Your project must focus on developing an intelligent application utilizing LLMs to automate, augment, or resolve specific vulnerabilities, threats, or operations within cybersecurity. Accepted implementation scopes include, but are not limited to:

- **Phishing & Social Engineering Detection**: Developing fine-tuned or prompt-engineered LLM pipelines to detect complex phishing contexts across emails, communication platforms, or SMS text channels.

- **Automated Malware & Code Vulnerability Analysis**: Using LLMs for reverse-engineering assistance, static code vulnerability hunting, malware behavior summarization, or explanatory decompilation.

- **Threat Detection & Log Auditing**: Parsing, normalizing, and correlating massive volumes of system/network logs (SIEM, firewall, EDR) to pinpoint anomalies, trace active threat actors, and extract threat intelligence indicators automatically.

- **Incident Response & Playbook Generation**: Creating conversational or autonomous orchestration agents that parse security alerts and automatically generate structured, step-by-step mitigation workflows.
